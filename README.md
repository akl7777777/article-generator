# 专业文章生成器

一个基于React的专业文章生成Web应用。

## 功能特点

- 主题选择和验证
- 大纲自动生成
- 文章内容生成
- 实时预览
- 编辑器支持
- 响应式设计

## 技术栈

- React
- Tailwind CSS
- shadcn/ui组件库

## 开始使用

1. 克隆仓库
```bash
git clone https://github.com/yourusername/article-generator.git
```

2. 安装依赖
```bash
cd article-generator
npm install
```

3. 启动开发服务器
```bash
npm run dev
```
